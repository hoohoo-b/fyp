Find under meta/ the following files:

1)gold.txt: Athlete's name -> Set of images for this athlete(.jpg extension removed)
2)graph.txt: (initial evidence) Image Pair -> Probability of being the same entity
3)answers.txt: ("experts" answers) Images Pair -> Set of answers for this pair (1->SAME, 0->DIFFERENT)
("experts" were mTurk workers with qualifications that were used for "difficult" image pairs)
